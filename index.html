Mercato Nova

Ce projet utilise PHP + SQL, donc il ne peut pas fonctionner complètement en ouvrant seulement ce fichier HTML.

Le vrai accueil du site est index.php. Pour le lancer, mets le dossier mercato-nova-php dans htdocs si tu utilises XAMPP, puis ouvre :

http://localhost/mercato-nova-php/index.php

À faire :
1. Lancer Apache et MySQL dans XAMPP.
2. Créer une base mercato_nova dans phpMyAdmin.
3. Importer database.sql.
4. Ouvrir le lien localhost ci-dessus.
Ouvrir index.php

Si tu cliques ici sans serveur PHP, le navigateur affichera le code ou ne lancera pas le site correctement.
